How to use this converter:

  • Click the upload area or drag & drop an image (max 1MB)
  • Check the file size - it must be 1MB or less
  • Choose output format: Data URL (for img.src) or Raw Base64
  • Click "Convert to Base64" to generate the encoded string
  • Use the "Copy to Clipboard" button to copy the result
  • Use "Reset" to start over with a new image
  • Data URLs can be directly used in HTML: <img src="data:image/jpeg;base64,...">
